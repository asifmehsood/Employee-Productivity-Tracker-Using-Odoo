Overview

The Employee Productivity Tracker is a complete Odoo module designed to monitor, track, and analyze employee productivity in real-time. It provides automated task timing, screenshot capture, app usage monitoring, and comprehensive reporting capabilities for managers.

Configuration Options

System Idle Timeout
Minutes of inactivity before automatically pausing timer (default: 15)
Screenshot Interval
Minutes between automatic screenshot captures (default: 10)
Screenshot Retention
Days to keep screenshots before auto-deletion (default: 30)
Activity Log Retention
Days to keep activity logs before auto-deletion (default: 90)
Restricted Applications
Comma-separated list of apps to detect and pause on detection
Notification Settings
Enable/disable notifications for managers on specific events

Database Tables

The module creates the following database tables:

  • productivity_task - Main task tracking table
  • screenshot_log - Screenshot storage
  • activity_log - Activity event logs
  • app_usage_log - Application usage tracking
  • productivity_report - Generated reports
  • productivity_config - Configuration storage

Security and Access Control

Access is controlled through Odoo's built-in security model:

  • Record Rules: Employees can only see their own data
  • Module Permissions: Defined in security/ir.model.access.csv
  • Group-Based Access: Different permissions for Employees, Managers, Admins
  • Data Encryption: Screenshots stored as binary attachments with access control

Performance Considerations

  • Screenshots are automatically compressed to configured size limit
  • Old data is automatically cleaned based on retention settings
  • Indexes on frequently queried fields for fast access
  • Configuration is cached to reduce database queries
  • Suitable for organizations with 100-1000+ employees
